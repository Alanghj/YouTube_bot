# YouTube_bot

## Put a random song to play, on Youtube.

### Virtual environment and Libraries in Linux/Windows:

```python 
Linux:
  python3 -m venv env
  source ./env/bin/activate
  pip3 install -r requirements.txt
  deactivate
  
Windows:
  python3 -m venv env
  .\env\Scripts\activate
  pip3 install -r requirements.txt
  deactivate
```
